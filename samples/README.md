# Sample tracks

A test track so the Media Player's Open dialog isn't empty on first use.
Plebian-OS's `apps/amp.py` copies it into `~/Music` when the player launches.

## ode-to-joy.ogg
Beethoven's "Ode to Joy" (Symphony No. 9, 4th movement) — the **composition is
public domain**. This audio is a **synthesized rendering generated from scratch**
(a simple additive-synthesis piano tone), not a recording of any performer, and
is released to the **public domain (CC0)**. Free to use, copy, and redistribute.
